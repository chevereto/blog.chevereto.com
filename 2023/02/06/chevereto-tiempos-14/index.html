# 🚀 Chevereto v4.0.7 released

Patch release 4.0.7 fixes several bugs which makes it the most stable release for 4.0 cycle. It also includes user favicon and comments tab.

Review Chevereto v4.0.7 release notes for a quick summary.

# User avatar favicon

Chevereto v4.0.7 will display for logged users their avatar as website favicon. This will make your website to stand taller among other tabs as it will show user's avatar instead of the website icon.

# Comments tab on images

In Chevereto v4.0.7 we are introducing a dedicated comments tab for images, with this new feature you'll be able to enhance the interactivity and community experience on your images as comments will be now on its own tab.

For comments you can use Disqus or any third-party comments provider.

# Bugs

Chevereto patch v4.0.7 fixes the following bugs:

  • Bulk importer error
  • Undefined property: stdClass::$rating_letter
  • There's no close button in the upload window
  • V3 update to V4.0.6 failed
  • Users cannot set call to action buttons
  • Scrolling to the bottom of a password-protected album when "endless scrolling" is enabled results in an error
  • Using Disqus leads to "something went wrong"
  • CDN CORS issue
  • Undefined array key "album"
  • Name undefined when upload as guest
  • Bug on the Register-Page
  • Setting the color for the call to action buttons does not work
  • CSS-user-Bug
  • Error exception

# 👍 Requests for changes (RFC)

We got the following requests for changes. Want to suggest yours? Feel free to share your thoughts and ideas.

  • Add user location on profile
  • Pin images in public mode
  • Add "_blank" checkbox to "call to action button"
  • External Storage SFTP with SSH Keys and / or FTP with TLS/SSL explicit encryption
  • Indicator for content to be moderated
  • Loading indicator for images/media that are still being downloaded
  • Thumb Size
  • Add call to action for user profiles
  • Add call to action settings
  • Moderate or history for call to action, avatar, background, username, email change
  • Need Tags system
  • Configurable display (images OR albums) for user profile
  • Wasabi and Storj as External-Storage
  • Pin-This (Pinterest Button)
