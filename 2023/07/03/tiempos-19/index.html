# 🛍️ Chevereto for Real Estate

Chevereto can now be used as a product gallery for your Real Estate business. We have prepared a complete guide on how to Real Estate with Chevereto which you can follow along to achieve this with no sweat.

# 👍 Requests for changes (RFC)

We got the following requests for changes. Want to suggest yours? Feel free to share your thoughts and ideas.

  • LINE login and sharing
  • Allow multiple users to upload photos to the same album
  • Allow admins to search for users via email as advanced search option in global search
  • Swipe down to close image in list viewer on mobile/tablet
  • Swipe to next/prev on image view page
  • Let right and left mouse click be system default
  • AI processing pictures
  • Categories display order
